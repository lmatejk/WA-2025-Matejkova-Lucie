Tlačítka

Bootstrap umožňuje snadné stylování tlačítek pomocí tříd jako např. btn-primary 

            
                <button class="btn btn-primary">Primary button</button>
                <button class="btn btn-warning">Warning button</button>

Outline tlačítko

Při použití třídy btn-outline-primary se tlačítko vybarví pouze, když na něj najedeme kurzorem 

            
                <button class="btn btn-outline-primary">Primary button</button>
                <button class="btn btn-outline-warning">Warning button</button>

Vypnuté tlačítko

Pro vypnutí tlačítka přidáme disabled

            
                <button class="btn btn-primary" disabled>Primary button</button>
                <button class="btn btn-outline-warning" disabled>Warning button</button>

Velikost tlačítek

Pro změnu velikosti tlačítka můžeme použít btn-lg nebo btn-sm 

            
                <button class="btn btn-primary btn-lg">Primary button</button>
                <button class="btn btn-outline-warning btn-sm">Warning button</button>

Můj názor

Bootstrap je velmi užitečný nástroj pro tvorbu webových stránek, hlavně pokud chceme vytvořit něco rychle a zároveň kvalitně. Výrazně ulehčuje práci tím, že nemusíme psát vlastní CSS - místo toho stačí použít třídy a komponenty Bootstrapu. Velkou výhodou je také jeho dokumentace, která je přehledná a dá se v ní rychle najít to, co zrovna potřebujeme.

Myslím, že celý Boostrap nejlépe vystihuje to, co mají napsaný přímo na oficiální stránce:

Build fast, responsive sites with Bootstrap
Powerful, extensible, and feature-packed frontend toolkit. Build and customize with Sass, utilize the prebuilt grid system and components, and bring projects to life with powerful JavaScript plugins.

Právě díky předpřipraveným třídám a komponentům může vytvoření jednoduché stránky zabrat jen několik minut. Tato stránka bude mít díky Boostrapu i funkčnost - nemusíme umět JavaScript a i přesto vytvoříme dobře fungující stránku.

Celkově si myslím, že každý, kdo vytváří jednodušší nebo středně složitý projekt, by měl Bootstrap určitě využít. Usnadní práci, ušetří čas a umožní soustředit se víc na obsah než na detaily ve stylování.
